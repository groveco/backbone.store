Class: exports

exports(optionsopt)

Adapter which works with data over HTTP, based on the options that are passed to its constructor. This class is responsible for any CRUD operations that need to be carried out with your JSON:API service from within the Backbone.Store.

 (()=>{}) A method to be executed on response to manipulate or handle the given response. A use case of this would be to add generic behavior or error reporting for certain status codes.
